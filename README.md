# Custom CRM for Freelancers

A lightweight CRM designed specifically for independent workers, focusing on lead tracking, project status, and communication history.

## Features
- **Lead Pipeline:** Visualize and manage your sales pipeline from initial contact to won projects.
- **Contact Management:** Keep track of key stakeholders and their roles in various companies.
- **Task Tracking:** Integrated task list to ensure you never miss a deadline or follow-up.
- **Business Insights:** Real-time stats on pipeline value and conversion rates.

## Setup

### App (Next.js)
1. Go to `root`
2. Run `npm install`
3. Run `npm run dev`

Note: This project is a frontend-only demonstration with mock data integration in `lib/crm-data.ts`.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Lucide React
- **Data Integration:** Mock API integration for high-performance UI demonstration.
