# PRD: Custom CRM for Freelancers

## 1. Project Overview
A lightweight CRM designed specifically for independent workers, focusing on lead tracking, project status, and communication history.

## 2. Target Audience
- Independent Developers
- Freelance Designers
- Solopreneurs

## 3. Key Features
- **Lead Pipeline:** Visualize leads from initial contact to project completion.
- **Contact Management:** Centralized database for client details and history.
- **Task Management:** Integrate tasks and deadlines directly into client profiles.
- **Notes and File Uploads:** Keep all project-related information in one place.

## 4. Tech Stack (CV Reference)
- **Frontend:** Next.js and Tailwind CSS (Simple, responsive interface).
- **Backend:** Node.js with a secure REST API.
- **Database:** Supabase for authentication and real-time data storage.
- **UI Library:** Shadcn/ui for modern and clean components.

## 5. Roadmap
- **MVP:** Simple contact and lead tracking system.
- **V1:** Task management and file upload support.
- **V2:** Calendar integration and automated lead follow-ups.
